{% trans "How to get access to the network" %}

{% trans "WARNING" %}: {% trans "this page will not be available after you have downloaded certificate and configuration files" %}

{% blocktrans %}
  • Download OpenVPN software
    • Windows version: http://swupdate.openvpn.net/community/releases/openvpn-2.2.0-install.exe
    • GNU/Linux version is usually available in the system software package manager (p.e: Ubuntu Software Center)
    • If not, you can install it by hand the GNU/Linux version from http://swupdate.openvpn.net/community/releases/openvpn-2.2.0.tar.gz
  • Run as administrator the downloaded file and install the OpenVPN software
    • {% endblocktrans %}
  • {% trans "Download certificate and configuration files" %}
    • {% blocktrans %}
  • Extract configuration files in the OpenVPN configuration folder
    • Windows default is C:\Program Files\OpenVPN\config
    • GNU/Linux (Ubuntu, Debian) default is /etc/openvpn/
  • Start the OpenVPN service:
    • In Windows right-click on OpenVPN GUI icon and select RUN AS ADMINISTRATOR
      • Then 2 computers will appear in the bottom-right of the screen, in the notification area
      • Right-click on those 2 computers and select Connect
      • Wait that computers become green and that connection succeed message is shown
    • In GNU/Linux (Ubuntu, Debian) start the service typing in terminal: service openvpn start
  • Open Mozilla Firefox or Google Chrome browsers and go to {{ VPN_HOME_PAGE }}
{% endblocktrans %}
